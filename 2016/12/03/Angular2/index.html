Angular

功能类工具方法

version ：获取你当前angular 版本对象

equals：和jquery中的equals 非常相似 用于比较

注意：

var a = NaN;  var b = NaN;  
var result = angular.equals(a,b);
console.log(result);  --   true

forEach：用于遍历数据

//value 和 index 顺序不能错
angular.forEach(c,function(value,index){

})

fromJson : 将字符串转换成json对象 相当于jquery中的parse

toJson ：将json对象转换成字符串

全局类方法

run 方法是专门用来设置全局属性的

它也只能设定$rootScope 不能设定$scope 作用域是当前的m1模块下

angular中的过滤器

angular中使用过滤器需要一个管道符号 |

angular中过滤器 使用功能很强大 可以自定义过滤器

angular中有7个自带的过滤器 还可以自定义过滤

1.将数据过滤为金钱类型

{{ text | currency：'￥'}}

currency – 默认$

2.将数据过滤成数字格式

{{ text | number：3 }}   -- 截取几位小数

3.将数据转换成大小写

$scope.info = $filter("uppercase")("xieling");   --  lowercase

4.美化json数据格式，没什么实际作用，就是为了让数据方便观看

{{ text | json}}

5.将数据进行排序

{{ text | orderBy："age" }}

6.将数据进行截取

{{ text | limitTo：4 }}  -- 截取4位

7.将数据进行过滤

{{ text | filter：'blue' }}  --过滤留下蓝色

8.angular 自定义过滤器

在m1这个module 下添加 一个 m1.filter()

m1.filter("firstUpper",function( ){   //自定义过滤器的名称  回调函数
 return function(str){   //传需要过滤的信息       自定义了一个首字母大写的过滤器
      return str.charAt(0).toUpperCase( ) + str.substring( 1 );      
}
})
然后在下面用这个自定义过滤器
{{ text | firstUpper }}

ng-repeat 循环指令

用于循环遍历的 说白了 就是帮你去拼接字符串

####关于ng-repeat 其他的一些参数

$index – 指你每个对应元素的下标 索引

$first – 返回的是布尔类型 若为第一个 则为真 其余为假

$last – 返回的是布尔类型 若为最后一个 则为真 其余为假

$middle – 返回的是布尔类型 若为中间 则为真 其余为假

$even – 返回的是布尔类型 若为偶数 则为真

$odd – 返回的是布尔类型 若为奇数 则为真

ng-repeat-start

ng-repeat-end

–避免在外面嵌套一个div 相当于微信小程序的block

如果dataList这个数组有重复的值[2,2,2,3,4,5],在低版本angular中使用data in dataList会报错 换成

data in dataList track by $index

特殊事件指令

ng-selected : 被选中事件指令

ng-change：chang事件触发和jquery中change事件的触发完全一致

ng-copy：复制或选中事件时触发

ng-cut：剪切时触发事件指令

ng-paste：粘贴时触发指令

状态类型指令

disabled readOnly checked

样式指令和传参指令

模块依赖引入方式

var m1 = angular.module(”app”,[ngSanitize]); ng+插件名称(首字母大写)

单个参数的传递

<p ng-bind="text"></p>

多个参数的传递

<p ng-bind-template="{{text1}},{{text2}}"></p>

传递标签(需要插件)

<p ng-bind-html="info"></p>

样式指令

<p ng-style="color">这个color指controller下定义的变量</p>

<p ng-class="{color:true}">这个color指class类名 单引号</p>

重写标签类型指令

ng-options ： 用于给你生成option 必须和 ng-model 一起使用

自定义指令

在angular中 除了它自带的一些指令外 还可以自定义

用m1.directive(“参数指令名”，回调)
